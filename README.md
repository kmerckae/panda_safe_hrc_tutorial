# panda_constrained_control_tutorial

## Getting started:

* install python3:
  ``` bash
  sudo apt install python3-pip
  ```


* install sphinx and read the doc theme with python3:
  ``` bash
  python3 -m pip install sphinx
  python3 -m pip install sphinx_rtd_theme
  ```
